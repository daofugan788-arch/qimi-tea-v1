# Hammer Commerce Agent V0.3

个人使用的移动端 AI 电商智能体。V0.3 完成 Sprint 03 选品助手：不仅能分析单个商品，还能从商品库横向对比候选项，给出优先测试顺序。

## V0.3 已完成

- 新增“选品对比”手机端入口
- 从 Products 商品库手动选择 2 个或更多候选商品
- `SelectionPrepareTool` 整理候选数据
- `ProductCompareTool` 按评分、利润率、单件利润排序
- 自动标记优先测试、备选、后备和淘汰商品
- 输出选品对比报告
- 生成首批测试数量、对照商品和 3 天数据验证计划
- 对比任务保存到任务历史
- 不调用或虚构实时平台数据

## V0.2 保留能力

- 手机端商品信息输入：名称、采购价、售价、运费、平台费用、备注
- `ProfitCalculatorTool` 利润计算工具
- 毛利润、销售利润、利润率、保本价和建议售价
- 商品 100 分评分
- 评分权重：利润 30%、需求 25%、竞争 20%、售后安全 15%、运输 10%
- 推荐等级：A / B / C / D
- 成本分析、风险判断、销售建议
- 商业分析报告
- Products 本机商品库
- 保留 V0.1 目标任务、Planner、Executor、Tool Registry 和任务历史
- PWA 安装与离线缓存基础

## 商品分析流程

```text
输入商品信息
  → 创建 PRODUCT_ANALYSIS 任务
  → Agent Planner 拆解五个步骤
  → 校验商品数据
  → ProfitCalculatorTool 计算利润
  → ProductScoreTool 五维评分
  → 生成商业分析报告
  → 保存到 Products 商品库
```

## 选品对比流程

```text
从商品库选择候选项
  → 创建 PRODUCT_COMPARISON 任务
  → 读取已保存的成本、利润和评分
  → 按评分、利润率、单件利润排序
  → 判断优先测试、备选或淘汰
  → 输出选品对比报告和测试顺序
```

## 利润公式

```text
毛利润 = 售价 - 采购价 - 运费
销售利润 = 毛利润 - 平台费用
利润率 = 销售利润 ÷ 售价 × 100%
最低成交价 = 采购价 + 运费 + 平台费用
```

## 当前边界

需求、竞争、售后和运输评分目前依据商品名称、备注与成本结构进行规则初评，尚未接入实时平台数据。V0.3 的排名只比较商品库中的已录入数据，系统不会虚构销量、搜索热度、货源或市场价格。

本版本不包含自动发布闲鱼、自动登录平台、自动下单、支付、库存或大型后台。
