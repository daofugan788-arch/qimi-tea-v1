# Hammer AI V0.6

面向普通微信卖货商家的移动端网页版内容助手。

首页价值说明：上传商品，1分钟生成朋友圈、小红书、短视频卖货内容。

核心流程：

```text
打开网页 → 填写商品 → 选择平台 → 生成内容 → 一键复制
```

## V0.6 付费验证

- 第一次内容生成免费，不阻挡新商家体验
- 第二次生成前展示商家内测方案
- 默认内测价为 19.9 元/月，可在 `config.js` 修改
- 支持手机分享或复制开通申请
- 本地记录生成、复制、方案查看和开通申请次数
- 暂不接账号、数据库和复杂支付系统

## 验收目标

- 陌生商家 5 秒内知道产品用途
- 不需要注册或学习提示词
- 1 分钟内完成第一次内容生成
- 结果可以一键复制发布
- 手机浏览器直接打开使用

## 支持内容

- 微信朋友圈成交文案
- 小红书标题与体验分享
- 抖音 30 秒口播脚本
- 淘宝详情卖点结构

## 本地运行

ES Modules 需要通过 HTTP 打开：

```bash
python3 -m http.server 8787
```

然后访问：

```text
http://127.0.0.1:8787
```

## Vercel 部署

将本目录设为 Vercel 的 Root Directory，Framework Preset 选择 Other，保持默认构建设置即可上线。

## AI 配置

默认使用本地模板，打开即可体验，商品资料不会上传。

`config.js` 保留 OpenAI 兼容接口配置。正式上线时必须通过服务端代理保存 API Key，不要把真实密钥写入静态网页或公开仓库。

## 冻结边界

本版本不包含自动发布微信、抖音账号管理、视频剪辑、AI 管家、商城系统和用户社区。
