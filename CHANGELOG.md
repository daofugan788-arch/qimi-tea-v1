# 更新记录

## V2.0.0 - 2026-07-14

- 新增本地 `IntentParser`，使用关键词和正则识别固定自动化指令
- 新增 `ActionPlanner`、`ActionValidator` 与统一 Action JSON 格式
- 新增 `ActionExecutor`，只执行网页内动作；Android 系统动作标记为需要外部执行器
- 新增 `ExecutionLogger` 与 `AutomationRepository`，本地保存流程、快捷命令和执行历史
- 新增“自动化”页面、Action 步骤卡片、风险提示、确认界面、取消按钮、日志和常用模板
- 新增 LOW、MEDIUM、HIGH 三级风险策略；MEDIUM 必须确认，HIGH 当前版本全部阻止
- 新增 Termux 命令白名单校验，禁止危险命令、管道执行、敏感目录读取、上传和网络扫描
- “解压最新暮曦 ZIP”只生成安全人工步骤，不直接控制 Android 文件管理器
- 新增自动化自测脚本，并更新 PWA 缓存到 V2.0
- 保留 V1.3 聊天、语音、记忆、设置、本地连续对话和远程模型预留模块

## V1.3.0 - 2026-07-14

- 新增本地基础意图分类，并按问候、时间、日期、身份、帮助、记忆、设置和情绪等类型回复
- 从已有聊天记录恢复上一轮意图和话题，支持重新打开应用后继续本地对话
- 新增时间与日期、帮助、设置、语音、聊天记录、长期记忆和情绪类连续追问
- 新增“刚才说什么”“还有呢”“在哪里”“那你呢”等上下文回复
- 继续轮换自然兜底回复，避免固定重复内容
- 保留现有 UI、语音输入、语音播报、聊天记录、长期记忆、设置及远程模型模块

## V1.2.0 - 2026-07-14

- 重写本地对话规则，删除重复的固定回复
- 新增问候、时间、日期、身份、帮助、记忆、设置等常见问题识别
- 新增本地语音、聊天记录、版本和情绪相关回复
- 无法理解时自动轮换自然兜底回复，避免连续重复同一句
- 保留远程模型模块，但本次未新增或修改远程接入能力

## V1.1.0 - 2026-07-14

- 新增统一 OpenAI 兼容 AI Provider
- 新增 Base URL、API Key、模型名称本地配置
- 兼容 OpenAI、DeepSeek、豆包、通义千问等 Chat Completions 接口
- 新增 ConversationManager 上下文管理与 Token 截断预留
- 新增 ChatRepository 本地聊天仓库
- 聊天页接入真实模型，支持加载、错误提示和重新发送
- 设置页新增保存配置和测试连接

## V1.0.0 - 2026-07-14

- 完成手机端 PWA 基础框架
- 完成文字聊天、语音输入和语音播报
- 完成长期记忆、本地聊天记录和数据备份
- 完成大模型安全代理接口预留
